## Conversion Step

All 9 source images were 1024×1024 JPG files.  
Script: inline Python using **Pillow + NumPy**

Processing pipeline per image:
1. Load JPG as RGBA
2. Sample all 4 corner pixels, average to detect background colour
3. Apply tolerance-based mask (±35 per channel) → set alpha=0 for bg pixels
4. Crop to bounding box of visible pixels (+ 4 px padding)
5. Save as PNG to `game/assets/images/`
